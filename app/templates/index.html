Shopping Service API v1.0.0

API-бэкенд сервиса покупки товаров для авторизованных пользователей.

С чего начать

Swagger UI Интерактивная документация — можно сразу вызывать эндпоинты ReDoc Удобное чтение полной документации API

Основные эндпоинты

POST/auth/register — регистрация нового пользователя POST/auth/login — вход по email или телефону, получение JWT GET/products/ — список активных товаров GET/cart/ — корзина текущего пользователя POST/cart/items — добавить товар в корзину

Служебные

GET/health — проверка работоспособности
